API Documentation

The Tweet Analyser API returns sentiment class of a given tweet. Expected resutls are 'Natural','Positive', or 'Negative'. To use this API, please follow the below instructions.

GET CLASS

To get the sentiment class of a tweet, a POST call to the following url can be used: https://flasktwitter.herokuapp.com/analyse/

URL

QUERY PARAMETERS

Field: tweet

Type: string

Description: a sample tweet or free text

Example:

https://flasktwitter.herokuapp.com/analyse/tweet=profit decrease

RESPONSE PARAMETERS

key: "results"

values: "neutral" or "positive" or "negative"
